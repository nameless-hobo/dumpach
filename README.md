# Dumpach
## SPA imageboard

Hosted board: [dumpach.anonymous.lv](https://dumpach.anonymous.lv)

Hosted on Digital Ocean

Front-end: React, Redux, React-Router, Redux Saga

Backend: Express, PostgreSQL
